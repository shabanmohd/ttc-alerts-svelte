Design Guidelines

  • Success variant: Use for positive "all clear" states (green border, green icon background)
  • Neutral variant: Use for filter results, search results, or informational states (dashed border, muted background)
  • Clickable states: Add cursor: pointer and hover effects when the empty state can trigger an action
  • Icon selection: Choose contextually relevant icons:
    • • Train/Subway for subway closures
    • • Calendar for date-filtered results
    • • Check Circle for "all clear" success states
    • • Alert Triangle for "no major alerts"
    • • Clock for "no minor alerts"
    • • Accessibility icon for accessibility-related empty states
    • • Route for route-related states
    • • Map Pin for stop-related states
    • • Bell for notification-related states
